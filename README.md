# mozlz4

## Introduction

Decompresses `*.jsonlz4` files used by Firefox. Tested with Clang on macOS and Linux.

## Getting Started

```sh
# Build
make CC=clang

# Run
./mozlz4 [.jsonlz4 file]
```
